Projetos Atuais e em Desenvolvimento de Mateus Bitar

1. Assistente de Portfólio Inteligente (RAG & GenAI)
Descrição: É o assistente com o qual você está interagindo agora. Trata-se de uma aplicação de Inteligência Artificial Generativa que utiliza a arquitetura RAG (Retrieval-Augmented Generation) para fornecer informações precisas sobre a trajetória do Mateus.
Tecnologias: Python, LangChain, Groq API (Llama 3.3 70B), Hugging Face Embeddings e ChromaDB.
Destaque: O sistema foi desenhado para eliminar alucinações da IA, garantindo que as respostas sejam baseadas estritamente nos dados reais de carreira e projetos do Mateus.
- Links do Projeto:
  * Repositório (GitHub): https://github.com/MateusBitar/LLM_repo
  * Aplicação (Deploy): https://mateusbitar-portfolio.streamlit.app/

2. Sistema de Clipping Jurídico e Newsletter (Legal-tech)
Descrição: Um serviço automatizado voltado para sistemas jurídicos que realiza a varredura, coleta e sumarização de atualizações legislativas e clipping de notícias jurídicas relevantes. O projeto visa entregar uma newsletter personalizada e filtrada por temas de interesse para advogados e escritórios de advocacia.
Tecnologias: Python (Scraping e Automação), Processamento de Linguagem Natural (NLP) para sumarização de textos e integração com APIs de envio de e-mail.
Status: Em desenvolvimento ativo, focado na escalabilidade do processamento de dados jurídicos e na precisão dos filtros de relevância.
Participação do Mateus: Atua como arquiteto do sistema e desenvolvedor principal, unindo conhecimentos de automação (RPA) e Ciência de Dados para tratar grandes volumes de informação.
