# Projeto de Destaque: Retenção Inteligente - Previsão de Churn com XGBoost

- O Desafio / Demanda: Em mercados competitivos (como o de telecomunicações), reter clientes é significativamente mais barato do que adquirir novos. O problema de negócio era desenvolver um pipeline de dados e um modelo preditivo capaz de identificar precocemente quais clientes possuíam maior probabilidade de evasão (churn), permitindo ações proativas de retenção.
- O que o Mateus fez (Ações e Arquitetura): Construiu uma aplicação completa de Machine Learning de ponta a ponta.
  1. Análise Exploratória (EDA): Mapeou os gatilhos de evasão, descobrindo que contratos mensais e o serviço de Fibra Óptica eram os maiores ofensores, enquanto o tempo de casa (tenure) atuava como escudo de retenção.
  2. Engenharia e Limpeza: Tratou inconsistências e aplicou One-Hot Encoding em variáveis categóricas.
  3. Modelagem Avançada: Como o dataset era altamente desbalanceado, aplicou a técnica SMOTE (Synthetic Minority Over-sampling Technique) para forçar o aprendizado estrutural da classe minoritária.
  4. Deploy: Construiu uma aplicação web interativa exportando o modelo (Joblib) para que equipes de vendas fizessem simulações visuais em tempo real.
- Tecnologias Utilizadas: Python, Pandas, Numpy. Para Machine Learning: Scikit-Learn, XGBoost (Extreme Gradient Boosting) e Imbalanced-learn (SMOTE). Para deploy e visualização: Streamlit, Plotly, Matplotlib e Seaborn.
- Entregas e Resultados (O ROI do Modelo): A substituição de um modelo baseline (Random Forest, que detectava 46%) pelo XGBoost aliado ao SMOTE fez o Recall saltar para 64%. O modelo conseguiu detectar 39% a mais de clientes em risco real de cancelamento. A entrega do app em Streamlit gerou alto valor tático, permitindo que a empresa salve centenas de assinaturas mensalmente com ofertas estratégicas.
