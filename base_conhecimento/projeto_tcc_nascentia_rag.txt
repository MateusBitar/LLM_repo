Projeto: Chatbot RAG Nascentia (Trabalho de Conclusão de Curso - TCC)
- Links do Projeto:
  * Repositório (GitHub): https://github.com/FelipeYoshidaCEUB/chatbot-ceub

Descrição Geral: O projeto Nascentia é um assistente virtual baseado em Inteligência Artificial desenvolvido para auxiliar na consulta de informações acadêmicas e institucionais. A aplicação utiliza a arquitetura RAG (Retrieval-Augmented Generation) para buscar contextos precisos em uma base de documentos antes de gerar as respostas, minimizando alucinações. O sistema foi construído integrando modelos de linguagem open-source, processamento de linguagem natural (NLP) com Hugging Face e banco de dados vetorial (ChromaDB), sendo orquestrado pelo framework LangChain.

Participação do Mateus: Este projeto foi desenvolvido inteiramente em grupo como o TCC da graduação. Mateus Bitar atuou como um dos desenvolvedores, contribuindo ativamente e de forma colaborativa em diversas frentes do projeto, ajudando a equipe a integrar os módulos da aplicação e entregar a solução completa, desde a estruturação dos dados até a conexão com o LLM.
